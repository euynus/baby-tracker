## 2024-03-02 - Icon-Only Button Accessibility\n**Learning:** Swift UI buttons using systemImage directly (e.g., Image(systemName: "chevron.left")) within a Button action need explicit .accessibilityLabel for VoiceOver to announce them correctly. Without it, the screen reader may just announce "Button" or the unlocalized system image name.\n**Action:** Add .accessibilityLabel("Localized Descriptive Text") to icon-only buttons like month selectors to improve navigation for visually impaired users.
